# Plot method for istat_forecast objects

Plot method for istat_forecast objects

## Usage

``` r
# S3 method for class 'istat_forecast'
plot(x, include_ensemble = TRUE, include_intervals = TRUE, ...)
```

## Arguments

- x:

  An istat_forecast object

- include_ensemble:

  Include ensemble forecast in plot (default TRUE)

- include_intervals:

  Include prediction intervals (default TRUE)

- ...:

  Additional arguments passed to plot
