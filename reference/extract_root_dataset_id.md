# Extract Root Dataset ID from Compound ID

Extracts the root dataset ID from compound ISTAT dataset IDs. For
example: "534_49_DF_DCSC_GI_ORE_10" -\> "534_49"

## Usage

``` r
extract_root_dataset_id(dataset_id)
```

## Arguments

- dataset_id:

  Character string with full dataset ID

## Value

Character string with root dataset ID
